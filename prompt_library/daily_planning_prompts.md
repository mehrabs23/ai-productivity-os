# Daily Planning Prompts

## Building a Personal Productivity System with AI — Prompt Library

---

## Prompt 1: Morning Planning Prompt

**Purpose:** Generate a structured, realistic daily plan by providing ChatGPT with your current context and priorities.

**When to use:** Every working morning, ideally within 30 minutes of starting work. Takes 10–15 minutes total.

**Ritual:** Open Notion dashboard → identify priorities → fill in this prompt → review and edit → commit to a Notion Daily Note.

---

```
I'm doing my morning planning for today. Please help me structure my day.

Today's date: [DATE]
Day of the week: [MONDAY / TUESDAY / ETC.]
Available productive hours today: [X hours]
Energy level right now (1 = very low, 5 = peak focus): [1–5]

Known constraints:
- [MEETING/CALL/OBLIGATION 1 — time and duration]
- [MEETING/CALL/OBLIGATION 2 — time and duration]
- [ANY HARD STOPS OR TRANSITIONS — e.g., "school pickup at 4pm"]

My top 3 priorities for today (from my Notion dashboard):
1. [PRIORITY 1 — task/project name + why it's top today]
2. [PRIORITY 2 — task/project name + why it's top today]
3. [PRIORITY 3 — task/project name + why it's top today]

One thing I've been avoiding that might need to happen today:
[TASK OR ACTION YOU'VE BEEN PUTTING OFF]

Please create a realistic, time-blocked daily plan that:
1. Assigns my top priorities to time slots that fit around my constraints
2. Groups similar tasks where possible to reduce context-switching
3. Builds in at least one buffer block (30+ minutes) for unexpected things
4. Places the most cognitively demanding work in my highest-energy period
5. Does NOT over-schedule — I should have some slack

Format the output as a clean time-block schedule. At the end, add one sentence noting the risk or assumption built into this plan.
```

## Prompt 2: Energy-Aware Planning Prompt

**Purpose:** An enhanced version of the Morning Planning Prompt that explicitly factors in energy patterns throughout the day.

**When to use:** Use this when you're making decisions about when to schedule specific types of work (deep work vs. meetings vs. admin).

---

```
I'm planning my day and want to match my task types to my energy pattern.

My typical energy pattern:
- Morning (7am–11am): [HIGH / MEDIUM / LOW focus capacity]
- Midday (11am–2pm): [HIGH / MEDIUM / LOW focus capacity]
- Afternoon (2pm–5pm): [HIGH / MEDIUM / LOW focus capacity]
- Evening (5pm–8pm, if working): [HIGH / MEDIUM / LOW focus capacity]

Today's date: [DATE]
Available hours: [X hours]

Tasks for today:
[LIST YOUR TASKS WITH ESTIMATED DURATION — one per line]

External constraints:
[LIST FIXED MEETINGS OR OBLIGATIONS WITH TIMES]

Please create a time-blocked plan that:
1. Places deep, creative, or complex cognitive work in my highest-energy period
2. Places calls, meetings, and admin in my lower-energy periods
3. Places review tasks and lighter work in transition periods (before or after meetings)
4. Respects my constraints
5. Includes a "worst case: drop this" note — if I only complete 60% of the plan, what should survive?
```

## Prompt 3: Minimal Day Planning Prompt

**Purpose:** A lightweight version for days where you have limited time or energy. Creates a minimum viable plan for the day.

**When to use:** When you're overwhelmed, under-resourced, sick, or just need to protect the most important thing.

---

```
Today is a difficult day for focus. I need a minimal, realistic plan.

I have approximately [X] productive hours available today.

The single most important thing I absolutely must move forward today is:
[ONE TASK OR OUTCOME]

Secondary items (only if the above is complete):
[TASK 2]
[TASK 3 — optional]

Things I should NOT try to do today (so I don't set up false expectations):
[THINGS TO DEFER]

Please give me:
1. A simple time structure for today (not heavily detailed — just the essential blocks)
2. A suggested start time for the most important task
3. One sentence framing: "Today is a good day if I..."
```

## Prompt 4: Context-Switch Recovery Prompt

**Purpose:** After an interruption (unexpected meeting, urgent request, personal disruption), use this to re-orient and decide what to do next.

**When to use:** Anytime during the day when your planned flow has been disrupted and you need to recalibrate.

---

```
My planned day has been disrupted. Help me recalibrate.

Time now: [TIME]
Hours remaining in my work day: [X hours]

What the disruption was:
[DESCRIBE BRIEFLY — e.g., "urgent client call ran 2 hours", "had to handle a personal situation"]

What I had planned to do (that I haven't done yet):
[LIST REMAINING TASKS FROM YOUR MORNING PLAN]

What's still due today (non-negotiable):
[HARD DEADLINE ITEMS IF ANY]

Please give me:
1. A revised plan for the remaining [X] hours
2. A clear statement of what I should deprioritize or defer to tomorrow given the disruption
3. One sentence: "Given what happened, a realistic and honest finishing state for today is..."
```
