# Review Prompts

## Building a Personal Productivity System with AI — Prompt Library

---

## Prompt 1: Weekly Review Prompt

**Purpose:** Generate a structured summary of your week and actionable recommendations for the next week.

**When to use:** Day 4, Session 3 — as part of the weekly review process. Use every Friday thereafter.

---

```
I'm running my weekly review. Please help me process and summarise my week.

Week of: [DATE RANGE]

Completed work (from my Anti-To-Do Log):
[PASTE YOUR ANTI-TO-DO ENTRIES FOR THE WEEK — one per line]

Tasks that were planned but not completed:
[LIST WITH REASON: was it externally blocked, actively avoided, or deprioritised?]

Active projects — status:
[PROJECT 1]: [moved forward / stalled / blocked]
[PROJECT 2]: [moved forward / stalled / blocked]
[PROJECT 3]: [moved forward / stalled / blocked]

Attention allocation — actual vs. ideal:
[DOMAIN 1]: Ideal [%] → Actual [%]
[DOMAIN 2]: Ideal [%] → Actual [%]
[DOMAIN 3]: Ideal [%] → Actual [%]

One thing that felt unexpectedly hard this week:
[BRIEF DESCRIPTION]

One thing that went better than expected:
[BRIEF DESCRIPTION]

Please:
1. Summarise the week in 3 sentences — what was the real output, and what was the dominant pattern?
2. Identify the single most significant time/attention misalignment
3. Suggest one specific, actionable change for next week that would address the biggest pattern you see
4. Ask me one question that would help me plan next week with more intention
```

## Prompt 2: Anti-To-Do Summary Prompt

**Purpose:** Generate a weekly accomplishment statement from your Anti-To-Do entries. Useful for performance reviews, LinkedIn updates, or simply recognising your real work.

**When to use:** As part of the weekly review, or any time you want a clear summary of what you shipped.

---

```
Please generate a concise accomplishment summary from my Anti-To-Do entries for this week.

Anti-To-Do entries:
[PASTE ALL ENTRIES FROM THE WEEK]

Please:
1. Write a 2–3 sentence summary that captures the main output and impact of this week's work
2. Organise by category (Work / Learning / Admin / Personal) and list 3–5 highlights per category
3. Identify any pattern in the type of work I completed (e.g., "mostly delivery work with little strategic work")
4. Write one sentence I could use to describe this week's work to a colleague or manager

Format: clean, clear, professional. No generic filler language.
```

## Prompt 3: End-of-Day Reflection Prompt

**Purpose:** Close the mental loop on the day — summarise what happened, identify what needs to carry forward, and prepare for tomorrow.

**When to use:** As a supplement to the Daily Check-In template — use when you want a conversational debrief rather than a structured form.

---

```
I'm ending my work day and want to close the loop. Here's my day summary:

Date: [DATE]
What I planned to do today: [LIST YOUR MORNING PLAN OR TOP PRIORITIES]
What I actually completed: [LIST WHAT YOU DID — can paste from Anti-To-Do]
What didn't move: [TASKS THAT DIDN'T GET DONE]
Focus quality (1–10): [SCORE]
Energy level at end of day (1–5): [SCORE]

One unexpected thing that happened: [IF ANYTHING]

Please give me:
1. A 2-sentence honest summary of today — what was the real story?
2. One insight about today that I should carry into tomorrow
3. A suggested first task for tomorrow morning based on what didn't move today
4. One thing I should NOT do tomorrow that I tried to do today (pattern to avoid)
```

## Prompt 4: Project Status Review Prompt

**Purpose:** Generate a clear status summary for a specific project — useful before a meeting, milestone review, or as part of the weekly review.

**When to use:** Any time you need to articulate where a project stands, or if a project feels confusingly vague.

---

```
I need a clear status summary for one of my active projects.

Project name: [PROJECT NAME]
Goal it serves: [PARENT GOAL]
Original deadline: [DATE]
Key milestones completed: [LIST COMPLETED MILESTONES]
Key milestones remaining: [LIST UPCOMING MILESTONES WITH TARGET DATES]
Last meaningful progress made: [DATE AND WHAT HAPPENED]
Current blockers (if any): [DESCRIBE BLOCKERS]
Upcoming tasks (next 7 days): [LIST 2–3 TASKS]

Please:
1. Rate this project's health: 🟢 On track / 🟡 At risk / 🔴 Stalled — with a 1-sentence reason
2. Identify the single biggest risk to hitting the deadline
3. Suggest the one action that would most improve the project's outlook right now
4. Write a 3-sentence status summary I could share with a stakeholder or put in a weekly update
```
