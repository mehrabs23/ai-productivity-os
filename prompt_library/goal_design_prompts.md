# Goal Design Prompts

## Building a Personal Productivity System with AI — Prompt Library

---

### How to Use This File

Each prompt below is a template. Replace everything in `[SQUARE BRACKETS]` with your own context before submitting to ChatGPT.

Always review and edit the AI output — it is a first draft, not a final answer.

## Prompt 1: Goal Definition Prompt

**Purpose:** Transform a vague goal statement into a structured, measurable goal with success criteria.

**When to use:** Day 1, Session 2 — when defining your goals for the first time, or any time you want to sharpen an existing goal.

---

```
I want to define a personal or professional goal more clearly. Here is my raw goal statement:

[PASTE YOUR RAW GOAL STATEMENT HERE — it can be vague, that's fine]

Please help me structure this into a well-defined goal by doing the following:

1. Rewrite the goal as a specific outcome statement (what changes in the world, not what I'll do — outcome-focused, not activity-focused)

2. Suggest 3–5 measurable success criteria — specific, observable statements that would tell anyone whether this goal has been achieved. At least one should be quantitative.

3. Identify 1–2 potential pitfalls or obstacles that might prevent achieving this goal based on how I've described it.

4. Ask me 2–3 clarifying questions that would help me sharpen the goal further, if needed.

Format the output clearly with headings for each section.

My context (add anything relevant): [DESCRIBE YOUR SITUATION, ROLE, TIME CONSTRAINTS, ETC.]
```

## Prompt 2: Goal Portfolio Review Prompt

**Purpose:** Review a set of existing goals and identify conflicts, gaps, and prioritization issues.

**When to use:** When you have 3+ goals and want to sense-check whether they're coherent as a portfolio, or during a retrospective.

---

```
I have the following active goals. Please review them as a portfolio and give me honest feedback:

[LIST YOUR GOALS — one per line, with their deadlines if applicable]

For each goal:
1. Classify it as: Professional / Personal / Learning / Financial / Health / Other
2. Rate the specificity of the success criteria on a scale of 1–3 (3 = clearly measurable)

For the portfolio as a whole:
3. Identify any goals that appear to conflict with each other (time, attention, or resources)
4. Identify any important life area that seems underrepresented
5. Suggest which 1–2 goals I should prioritise if I had to choose
6. Flag any goal that looks more like a project or a task (not truly a goal-level outcome)

My context: [DESCRIBE YOUR CURRENT LIFE SITUATION — WORK, COMMITMENTS, TIME AVAILABLE, ETC.]
```

## Prompt 3: Goal Narration Prompt

**Purpose:** Turn a structured goal into a short motivational narrative — useful for staying connected to the "why" behind a goal.

**When to use:** Optional — use to create a 1-paragraph "goal story" that you read at the start of each Monday.

---

```
I have a goal with the following details:

Goal: [GOAL NAME]
Outcome: [OUTCOME STATEMENT]
Deadline: [DATE]
Success criteria:
- [CRITERION 1]
- [CRITERION 2]
- [CRITERION 3]

Please write a short motivational narrative (1 paragraph, 3–5 sentences) that:
1. Describes what my life or work looks like after this goal is achieved
2. Connects it to a meaningful personal reason (you can invent a plausible reason based on the goal — I'll edit it)
3. Uses specific, concrete language — not generic inspirational language

This will be something I read to reconnect with my goal when I lose motivation. Make it honest and grounded, not abstract or generic.
```

## Prompt 4: Goal Success Criteria Stress Test

**Purpose:** Challenge a goal's success criteria to ensure they're actually measurable and in your control.

**When to use:** After writing your first draft of success criteria — before entering them into your Notion database.

---

```
Please stress-test the following goal and its success criteria:

Goal: [GOAL NAME AND OUTCOME STATEMENT]
Success criteria:
- [CRITERION 1]
- [CRITERION 2]
- [CRITERION 3]

For each success criterion, evaluate:
1. Is this observable? (Can someone else verify it without asking you how you feel about it?)
2. Is this in my control? (Or does it depend heavily on external factors like other people's decisions or market conditions?)
3. Is this the right level — or is it too easy/too hard to count as real success?

Then suggest at least one revision to any criterion that fails these checks.
```
