# Student Handbook

## Building a Personal Productivity System with AI

---

Welcome to the course. By the time you finish Day 4, you will have built something real: a personalised AI-assisted productivity system you will actually use.

This handbook is your companion for all four days. Keep it open alongside your Notion workspace.

---

### What You're Building

Over four days, you'll assemble a complete **Personal Productivity Operating System** — a self-contained, AI-powered workspace that tracks your goals, organises your work, and gives you a daily ritual for planning and reflection.

**By end of Day 4, your system will include:**

| Component | What It Does |
|-----------|-------------|
| Goals Database | Captures your goals with specific success criteria |
| Projects Database | Organises your work into bounded deliverables, linked to goals |
| Task Database | Lists atomic actions, with priority and due dates |
| Milestone Map | Breaks projects into checkpoints with target dates |
| Anti-To-Do List | Logs what you've done — not just what's next |
| Work Session Log | Records your actual focused work blocks |
| Attention Budget | Compares ideal vs. actual time allocation across domains |
| Kanban Board | Visualises task flow through workflow stages |
| Master Dashboard | Aggregates everything into one navigable view |
| Streak Tracker | Monitors daily habits and consistency |
| Daily AI Planning Routine | A 10-min morning ritual with ChatGPT |
| Weekly Review | A 25-min Friday ritual for closing the loop |
| Retrospective | A monthly system health check |

### How Each Day Works

Each day follows the same rhythm:

1. **Opening (15 min):** Recap, questions, energy check
2. **Sessions (4 × 60 min):** Teaching → Demo → You build
3. **Closing (30 min):** Share what you built, reflect, preview tomorrow

**Your role in each session:**
- Use the **daily task sheet** (in `student_task_sheets/`) to guide your build tasks
- Use the **prompt library** when a session calls for AI input
- Complete the **daily check-in** at the end of each day

---

### Using the Prompt Library

The prompt library ([`prompt_library/`](../prompt_library/)) contains curated ChatGPT prompts for every task and stage of this course. Each file covers one category of work, and each prompt is a structured template — you fill in the placeholders between `[brackets]` with your real data.

**Prompt library files:**

| File | What it covers |
|------|----------------|
| `goal_design_prompts.md` | Goal definition, SMART conversion, feasibility check, long-term roadmap, goal portfolio review |
| `task_breakdown_prompts.md` | Brain dump → system, goal → project → task hierarchy, effort estimation, Notion-ready tables |
| `prioritization_prompts.md` | Task prioritization, impact/effort matrix, project ranking, overload triage, attention budget, weekly reset |
| `daily_planning_prompts.md` | Morning planning, energy-aware scheduling, minimal day, focus blocks, deadline-first planning |
| `review_prompts.md` | End-of-day reflection, stuckness diagnosis, weekly review, blocker diagnosis, progress update generator, retrospective |
| `system_optimization_prompts.md` | Life OS audit, dashboard audit, clutter removal, habit design, consistency repair, persona packs |

Each file also includes **persona-specific prompt variants** for students, freelancers, and content creators — labelled clearly within each file.

**Golden rules for using the prompt library:**
1. Always review and edit AI output before using it — the AI gives you a first draft, not a final answer
2. The more specific your input, the more useful the output — don't use vague placeholders like "my goals"
3. If a prompt produces output that feels generic, add more context to the prompt and retry

### Working With Your Real Data

This course only works with **your actual goals, projects, and tasks**. No fictional practice exercises. If you're worried about sharing something during the course, you don't have to — but do put it in your system. The system works in proportion to how honest you are with it.

---

### Daily Checklist

Use this at the start and end of each day:

**Start of day:**
- [ ] Open your Notion workspace
- [ ] Open ChatGPT in a separate tab
- [ ] Read the task sheet for today and identify the Minimum Viable Task
- [ ] Note: what is the one thing I most want to have built by end of today?

**End of day:**
- [ ] Complete the Daily Check-In template
- [ ] Update your Anti-To-Do with at least 2 entries from today
- [ ] Log today's work sessions (at least approximate)
- [ ] Preview tomorrow's task sheet

### Assessment and Capstone

The course ends with a **10-minute capstone presentation** on Day 4 afternoon. You will show your Notion workspace and walk an evaluator through your system, explaining the decisions you made.

**Read the full capstone instructions** in [`capstone_instructions.md`](capstone_instructions.md) — ideally by Day 3.

There are no tests or quizzes. Your work speaks for itself.

---

### Getting Help

**During the course:** Ask your instructor any time. There are no stupid questions — the most common questions are "I think I'm doing this wrong" (you're probably not) and "Is this specific enough?" (try to make it even more specific).

**After the course:** The prompt library, templates, and this handbook are yours to keep. The Notion system you built is yours. Use them.
